# CRUD_Node.JS
Create, Read, Update and Delete operations using Node.js

## Steps

- Download the source
- run Npm Install
- run make sure you have nodemon
- run nodemon index.js
- import the API collection which is names as CURD to POSTMAN
- Enjoy The Coding !!
